编辑HR信息

编号  :  C1

POST - http://api.597.com/company/hr
HR招聘者填写个人信息申请加入公司
请求参数
参数名 参数类型 必传 缺省值 描述
realname string Y 姓名
gender int Y 性别,1-男。2-女
attachment string N 头像
token string Y 登录产生的token
cname string Y 招聘者手动输入的公司名称
jobName string N hr职务
nameShow int N 称呼字段
weixinNO string N 微信号
motto string N 座右铭
返回值
加入已经存在的公司，返回如下值：
{
    "status": 1,
    "msg": "加入成功！",
    "isAuth":0,
    "cid": "加密的cid编号"
}
加入不存在的公司，返回如下值：
{
    "status": 2,
    “isAuth”:0,
    "msg": "请注册公司！"，
    "cname": "输入的公司名称！"
}
备注
注册已经存在的公司会返回一个 cid，进入加入已经存在的公司页面。注册的公司名称不存在，不会产生cid，进入注册新公司页面。
isAuth  实名认证标识：0未认证，1已认证
